# Round-UI for Unity

  Shader-based rounded UI component for Unity with precise hit detection.

![ScreenShot](https://github.com/berkencami/round-ui-unity/blob/main/Assets/Round-UI/sample.png)

  ## Features

  - Per-corner rounding control
  - Fill and Border modes
  - GPU-accelerated rendering
  - Accurate click detection on rounded areas
  - Works with Unity's standard UI system
  - Customizable outline with adjustable thickness and color
  - Gradient color effects (vertical, horizontal, diagonal) with configurable color stops
